# ManERP

实现主现进销存业务的一体化管理， 能最大限度地整合库存结构、减少滞销库存、刺激商品销售。在管可式上变被动管理为主动管理，自动化程度高，决策支持全面，能够准确及时地为决策者提供第一 手是企业长期信息化投资的理想产品
